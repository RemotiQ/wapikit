---
Title: Quick Start
Description: Learn how to quickly get started with WapiKit.
---

To get started using WapiKit you need to self host the application.

<Steps>
    <Step> 
    Install WapiKit
    <Note>WapiKit comes as a single binary application, can be installed easily on your local machine or server. Read this guide [here](/guide/installation-and-preparation/installation).</Note>
    </Step>
    <Step>
        Configure Whatsapp Business Account Settings. Check this guide [here](/guide/whatsapp-api-setup/configuring-wapikit).
    </Step>
     <Step>
         Create Contact List.
         <Note>Navigate to `/lists` and create a new list.</Note>
    </Step>
    <Step>
         Import Contacts.
         <Note>Navigate to `/contacts` and create a contact or bulk import the contact.</Note>
    </Step>
     <Step>
         Create Campaign.
         <Note>Navigate to `/campaigns` and create a new campaign.</Note>
    </Step>
    <Step>
         Send Campaign.
    </Step>
</Steps>
